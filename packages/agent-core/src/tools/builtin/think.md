Use the Think tool whenever you need to reason through a complex decision, plan a multi-step operation, or reflect on a constraint before taking action. Write a concise chain-of-thought in `thought`. The thought is recorded in the session so you can refer back to it, but it does **not** change files or run commands.

Prefer calling Think before:
- Editing multiple files that depend on each other.
- Choosing between several implementation approaches.
- Deciding whether to ask the user for clarification.
- Breaking a large request into smaller steps.

After recording your thought, proceed with the appropriate tools.
